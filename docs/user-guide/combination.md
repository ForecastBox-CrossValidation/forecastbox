# Forecast Combination

## Overview

Combining multiple forecasts typically outperforms individual models.

## Available Methods

| Method | Description |
|--------|-------------|
| mean | Simple average |
| median | Median forecast |
| inverse_mse | Inverse MSE weighting |
| ols | OLS regression weights |
| bma | Bayesian Model Averaging |
| stacking | Stacked generalization |
| optimal | Optimal combination (Bates-Granger) |
