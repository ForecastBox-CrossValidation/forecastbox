# MIDAS Models

## Mixed-Data Sampling

MIDAS models use high-frequency regressors to forecast low-frequency variables.

$$y_t^{(Q)} = \beta_0 + \beta_1 B(L^{1/m}; \theta) x_t^{(M)} + \epsilon_t$$
